# Asset Report Generator (Python Demo)

## Overview
A Python demo project that reads asset data from a CSV file and generates a simple asset status report.

## Purpose
This project demonstrates basic Python skills in file handling, CSV processing, data analysis, and report generation.

## Features
- Reads asset data from a CSV file
- Displays asset details
- Counts total assets
- Counts assets by status
- Generates a summary report

## Technologies
- Python
- CSV

## How to Run

```bash
py main.py
